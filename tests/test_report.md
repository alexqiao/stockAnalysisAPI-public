## 运行测试

### 环境要求
```bash
conda activate tradingagents
```

### 运行所有测试
```bash
python -m pytest tests/ -v
```

### 运行特定测试
```bash
# 仅运行单元测试
python -m pytest tests/unit/ -v

# 仅运行集成测试
python -m pytest tests/integration/ -v

# 运行特定文件
python -m pytest tests/integration/test_auth_flow.py -v
```
